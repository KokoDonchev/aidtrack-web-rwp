API Endpoints

  1. Dashboard / API

Campaigns Endpoints

GET /v1/campaigns Get a list of all campaigns
GET v1/campaigns/id/campaign-id Get information about single campaign
POST v1/campaigns Create a new campaign

Shipments Endpoints

GET /v1/shipments Get a list of all shipments
GET v1/shipments/id/shipment-id Get information about single shipment
POST v1/shipments Create a new shipment

Products Endpoints

GET /v1/products Get a list of all products
GET v1/products/id/product-id Get information about single product
POST v1/products Create a new product
